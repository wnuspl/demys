# demys
Lightweight console text editor. Simplifies window/tab management, config/settings, no extra unused (by me) features.

---
### design goals

- Similar feel to vim
- Easier to multitask
- Easy to edit preferences (no googling)

### planned features

- Display file system (and open files from it)
- Dedicated settings page
- Hide/reveal labels/commands
- Simple syntax highlighting
- Color themes
